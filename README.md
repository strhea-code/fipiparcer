# fipiparcer

Скачивает задания из открытого банка ФИПИ и собирает редактируемые карточки
в файл Word (`.docx`) — для печати на A4 или раскладки по темам в самом Word.

- Текст и формулы — **редактируемые**, не картинки. Формулы — настоящие
  Office Math equations (то, что вставляет «Вставка → Уравнение» в Word).
- Каждая карточка — **отдельная таблица**. Можно перетащить, изменить
  размер, скопировать в другой документ — независимо от остальных.
- Сверху карточки — номер задачи (qid), снизу — «Ответ:» и плашка
  квадратиков для записи цифр ответа.
- Развёрнутые задачи кладутся в отдельный файл (без квадратиков).

## Что нужно для запуска

- macOS / Windows / Linux
- Python 3.11+
- Доступ в интернет (банк ФИПИ открыт только для российских IP)

## Установка (первый раз)

Открой Терминал в папке проекта. Все команды копировать и вставлять целиком.

```bash
# 1. Создать виртуальное окружение (изолированный Python для проекта)
python3 -m venv .venv

# 2. Активировать
source .venv/bin/activate           # macOS / Linux
# .venv\Scripts\activate            # Windows (без source)

# 3. Установить библиотеки
pip install -r requirements.txt

# 4. Скачать браузер для парсинга (~150 МБ, один раз)
playwright install chromium
```

После установки `.venv` и `chromium` живут на твоём компьютере — повторно
ставить не надо.

## Как запустить

```bash
# 20 заданий из ОГЭ-математики → data/output/oge_math_sample.docx
python fipiparcer.py sample-docx --n 20

# Открыть папку с результатом
open data/output         # macOS
# explorer data/output   # Windows
```

Через 30–60 секунд в `data/output/` появятся:

- `oge_math_sample.docx` — короткие задачи и задачи с выбором ответа
- `oge_math_sample-extended.docx` — развёрнутые (если такие попались)

Открывай в Word или Pages — карточки готовы к печати/правке.

## Другие банки заданий

По умолчанию — ОГЭ-математика. Другие банки указываются флагами:

```bash
# ЕГЭ профильная математика
python fipiparcer.py sample-docx \
    --host ege.fipi.ru \
    --proj AC437B34557F88EA4115D2F374B0A07B \
    --n 20

# Любой другой банк: открой нужный банк на fipi.ru,
# скопируй GUID из адресной строки и подставь в --proj
```

| Банк | host | proj |
|---|---|---|
| ОГЭ математика | `oge.fipi.ru` | `DE0E276E497AB3784C3FC4CC20248DC0` |
| ЕГЭ математика профильная | `ege.fipi.ru` | `AC437B34557F88EA4115D2F374B0A07B` |

GUID-ы других банков лежат на [fipi.ru/ege](https://fipi.ru/ege/otkrytyy-bank-zadaniy-ege) и [fipi.ru/oge](https://fipi.ru/oge/otkrytyy-bank-zadaniy-oge) — в адресной строке после `?proj=`.

## Все параметры

```
--host <hostname>      oge.fipi.ru (по умолчанию) или ege.fipi.ru
--proj <GUID>          Идентификатор банка из URL ФИПИ
--n <N>                Сколько заданий взять (по умолчанию 20)
--per-page <0|4|6>     Целевое количество карточек на A4 листе:
                       0 = высота по контенту (плотно, по умолчанию)
                       4 = минимум 9 см на карточку
                       6 = минимум 6 см на карточку
--out-dir <путь>       Папка результата (по умолчанию data/output)
--out-name <имя>       Префикс имён файлов (по умолчанию oge_math_sample)
```

## Что где лежит (если захочешь поправить)

| Файл | За что отвечает |
|---|---|
| `fipiparcer.py` | Командная строка. Если хочешь добавить новый флаг — сюда. |
| `extractor.py` | Открывает страницу ФИПИ через браузер, вынимает текст задач и формулы (в формате MathML), картинки. Если ФИПИ поменяет вёрстку — обновлять селекторы здесь (см. константу `EXTRACT_JS`). |
| `math_convert.py` | Конвертирует MathML-формулы в OMath (формат уравнений Word). |
| `docx_builder.py` | Собирает `.docx`. Размер карточки, ширина, поля, шрифт — здесь. Константы наверху файла: `CARD_WIDTH_CM`, `A4_LANDSCAPE_WIDTH_CM`, `PAGE_MARGIN_CM`. |
| `assets/answer_squares.png` | Плашка пустых квадратиков под слово «Ответ». Замени файл на свой, если хочешь другой вид. |
| `requirements.txt` | Список Python-библиотек (для `pip install`). |
| `samples/reference/` | Эталонные `.docx` — пример карточки и образец конечного вида. |

## Частые правки

**Сделать карточки шире/уже** — в `docx_builder.py` поменяй `CARD_WIDTH_CM = 13.0`.

**Сделать карточку другой высоты** — в `docx_builder.py` в словаре `CARD_HEIGHT_BY_PER_PAGE` поменяй значения. Например, `{0: None, 4: 8.5, 6: 5.5}`.

**Поменять плашку квадратиков** — замени файл `assets/answer_squares.png`.

**Шрифт заголовка / тела карточки** — в `docx_builder.py:_fill_card` строки с `Pt(10)` и `Pt(11)`.

**Игнорировать ещё какую-то служебную фразу** — в `extractor.py:clean_chunks` добавь в `SKIP_PATTERNS`.

**Сменить ОГЭ на ЕГЭ по умолчанию** — в `fipiparcer.py` константа `PROJ_OGE_MATH` + значение `--host` по умолчанию.

## Что делать, если ФИПИ обновил банк

Просто запусти команду снова — программа каждый раз идёт на сайт ФИПИ и вытаскивает то, что там сейчас. Старые файлы в `data/output/` перезапишутся.

Каждая задача идентифицируется внутри своим GUID (стабильный 32-символьный идентификатор, ФИПИ его не меняет даже при перетряхивании банка). Так что если ФИПИ переименует короткий qid — задача всё равно найдётся.

## Если что-то не запускается

- `playwright._impl._errors.TimeoutError` при `Page.goto` — нет интернета или ФИПИ упал. Подожди 5 минут, запусти снова.
- `ModuleNotFoundError: No module named 'playwright'` — забыли активировать `.venv`. Запусти `source .venv/bin/activate`.
- `Executable doesn't exist at ...chromium...` — не сделан `playwright install chromium`.

## Под капотом

1. Открываем `https://oge.fipi.ru/bank/questions.php?proj=...&init_filter_themes=1` через headless-браузер (Chromium).
2. ФИПИ грузит задания через JavaScript — ждём, пока появятся блоки `.qblock` (10 штук на страницу).
3. Из каждого блока вытаскиваем: `qid` (из `id="q..."`), `guid` (из скрытого input), тип ответа (короткий / выбор / развёрнутый), и **пошагово** — текст и формулы.
4. Формулы у ФИПИ хранятся как MathML (доступ через MathJax API). Конвертируем в OMath через библиотеку `mathml2omml`.
5. Идём по страницам пагинации (`&pagenum=2,3,...`) пока не наберём нужное количество или новые задачи кончатся.
6. Собираем `.docx`: альбомная A4 в две колонки, каждая карточка — отдельная таблица 1×1 шириной 13 см, текст редактируемый, формулы — настоящие уравнения Word.
